# 🌿 Compass Kinetics | 慧航自成

Welcome to the poetic repository of self-actualization. This space holds bilingual worksheets, visual metaphors, and coaching prompts designed to awaken clarity, creativity, and legacy.

## ✨ What’s Inside

- **Becoming by Design**: A worksheet for intentional transformation
- **Elder Reflection Prompts**: Legacy questions for wisdom keepers
- **Seasonal Archetypes**: Visual metaphors for life transitions
- **Prompt Generator**: Python script to create coaching questions
- **Bilingual Translations**: Mandarin versions for cultural resonance

## 🧭 Philosophy

Rooted in Eastern wisdom and systems thinking, this toolkit invites reflection, dialogue, and poetic branding. It is designed for coaches, educators, and lifelong learners.
